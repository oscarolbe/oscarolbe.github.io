Current Project

e-Matrix: EdTech Instructional Tracking Platform

e-matrix.at

I'm currently working at e-Matrix developing an innovative instructional-tracking software platform designed for enterprises, factories, and wide range of organizations, using Elixir, LiveView, and Phoenix framework. This EdTech solution addresses the critical need for structured training and skill development in technical environments.

The platform enables organizations to:

  • Create and manage comprehensive training programs for technical staff
  • Track employee progress through customized learning paths
  • Document competencies and certifications required for compliance
  • Analyze training effectiveness with detailed reporting tools
  • Streamline the onboarding process for new employees

This project has been particularly rewarding as it combines educational technology with industrial applications, creating a solution that directly impacts workforce development and operational efficiency in manufacturing and technical environments.
